# Tools yang Dibutuhkan (kalau pakai Android)

Termux

# Persiapan

```bash
pkg install git
pkg install php
```

# Instalasi

```bash
cd /sdcard
git clone --depth=1 https://github.com/mzaini30/quest-public
```

# Menjalankan

```bash
cd /sdcard/quest-public
php -S localhost:8067
```

Lalu setelah itu buka browser terus buka `http://localhost:8067`
